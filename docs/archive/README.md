## When to Archive Documentation

Documents should be moved to `/docs/archive/` when:
- ✅ All work described is 100% complete
- ✅ Document serves primarily as historical reference
- ✅ Current status is documented elsewhere
- ✅ Content would confuse users looking for current state

Documents should **NOT** be archived if:
- ❌ Work is partially complete
- ❌ Contains current configuration/setup instructions
- ❌ Referenced frequently in active work
- ❌ No replacement documentation exists
